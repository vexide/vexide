# vexide-graphics

A collection of V5 Brain display driver implementations for embedded rendering libraries.

![Maintained: no](https://img.shields.io/maintenance/no/2025)

> [!important]
> This crate is deprecated.
>
> You should consider using either
> [`vexide-embedded-graphics`](https://crates.io/crates/vexide-embedded-graphics) or
> [`vexide-slint`](https://crates.io/crates/vexide-slint) depending on which graphics library you
> prefer.
